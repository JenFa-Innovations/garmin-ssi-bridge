# 3D Prints

Enclosure for the Dive Transfer device.

## Parts

| Part | Material | Reason |
|---|---|---|
| Main body | ASA | UV and chemical resistance |
| Display bezel / lens ring | PETG | Clarity, easy to print |
| USB-C port seal | TPU 95A | Flexible dust cover |
| NFC lid insert | PETG | Holds NFC sticker flat |

## Print Settings

| Setting | Value |
|---|---|
| Layer height | 0.2mm |
| Walls | 3 |
| Infill | 20% gyroid |
| Support | Only on buildplate |
| ASA enclosure | Enclosed printer recommended (warping) |

## Assembly Notes

- Press-fit PCB into main body — 4 snap clips, no screws
- GC9A01 display sits flush in circular recess, held by bezel ring
- USB-C cutout accepts standard SMD receptacle footprint
- NFC sticker sits in shallow pocket on lid underside — no adhesive needed
